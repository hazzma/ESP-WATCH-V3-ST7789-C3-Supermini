# Debounce Logic

## Core Capabilities
- **Stable State Detection**: Memastikan perubahan status tombol bertahan selama minimal 20ms (atau nilai konfigurasi) sebelum dianggap sebagai input valid.
- **Glitch Removal**: Menghilangkan *noise* atau *spikes* elektrik yang terjadi saat kontak fisik tombol pertama kali bersentuhan.
- **Documentation**: Memberikan alasan teknis pada komentar kode mengenai pemilihan nilai debounce berdasarkan karakteristik tombol fisik.
